# Automation Stuff

A collection of automation scripts, utilities, and learning projects for DevOps, infrastructure management, and software development.

## Contents

### VMware/vSphere Tools
- **vsphere_ips/** - Scripts for managing vSphere VMs and IP addresses
- **VMware-ovftool** - OVF Tool bundles for VM deployment

### Container & Orchestration
- **docker/** - Docker configurations and utilities
- **kubectl-avi-vmware/** - Kubernetes kubectl configurations

### Automation Scripts
- **scripts/** - Python automation scripts for various tasks
- **globalprotect_script/** - GlobalProtect VPN automation
- **selenium_tp_link_load_balancer.py** - Selenium automation for TP-Link load balancer configuration

### Certificate Management
- **certs/** - Certificate management utilities
- **syslog_all_certs/** - Syslog certificate tools

### Learning Projects
- **golang_udemy/** - Go programming exercises (GoCasts)
- **golang_udemy_2/** - GORM database examples
- **protobuf-example-go/** - Protocol Buffers examples in Go
- **Gate/C_programming/** - C programming practice

### Other Tools
- **uwsgi-spooler/** - uWSGI spooler configurations
- **controller_customization_new/** - Controller customization scripts
- **dev-vm-files/** - Development VM configuration files
- **packtpub/** - PacktPub learning resources

## Technologies Used

- Python
- Go
- Docker
- Kubernetes
- VMware vSphere
- Selenium
- Protocol Buffers
